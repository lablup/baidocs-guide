# 어드모니션

어드모니션(콜아웃 또는 경고라고도 함)은 중요한 정보를 강조하는 특별한 스타일의 박스입니다.

## 기본 문법

디렉티브 문법에 세 개의 콜론 사용:

```markdown
:::type
내용이 여기에 들어갑니다
:::
```

## 사용 가능한 타입

### 노트

```markdown
:::note
일반 정보와 설명을 위한 노트 어드모니션입니다.
:::
```

:::note
일반 정보와 설명을 위한 노트 어드모니션입니다.
:::

### 정보

```markdown
:::info
정보성 메시지를 위한 정보 어드모니션입니다.
:::
```

:::info
정보성 메시지를 위한 정보 어드모니션입니다.
:::

### 팁

```markdown
:::tip
유용한 조언과 모범 사례를 위한 팁 어드모니션입니다.
:::
```

:::tip
유용한 조언과 모범 사례를 위한 팁 어드모니션입니다.
:::

### 성공

```markdown
:::success
성공적인 작업을 확인하기 위한 성공 어드모니션입니다.
:::
```

:::success
성공적인 작업을 확인하기 위한 성공 어드모니션입니다.
:::

### 경고

```markdown
:::warning
중요한 경고와 주의사항을 위한 경고 어드모니션입니다.
:::
```

:::warning
중요한 경고와 주의사항을 위한 경고 어드모니션입니다.
:::

### 위험

```markdown
:::danger
위험한 작업에 대한 치명적 경고를 위한 위험 어드모니션입니다.
:::
```

:::danger
위험한 작업에 대한 치명적 경고를 위한 위험 어드모니션입니다.
:::

## 색상 참조

| 타입 | 색상 | 사용 사례 |
|------|------|----------|
| **note** | 연한 파랑 | 일반 노트와 정보 |
| **info** | 청록색 | 정보성 콘텐츠 |
| **tip, success** | 녹색 | 유용한 조언, 성공적인 결과 |
| **warning** | 주황색 | 중요한 경고 |
| **danger** | 빨간색 | 치명적 경고, 오류 |

## 고급 사용법

### 포맷된 콘텐츠

어드모니션 내부에 마크다운 포맷팅 사용:

```markdown
:::note
**굵게**, *기울임*, 그리고 `코드`를 사용할 수 있습니다.

- 리스트도 작동
- 여러 항목

[링크](https://example.com)도 작동합니다!
:::
```

### 코드 블록

어드모니션에 코드 블록 포함:

```markdown
:::tip
개발 서버를 시작하는 방법:

\`\`\`bash
pnpm dev
\`\`\`
:::
```

# BaiDocs에 오신 것을 환영합니다

기술 문서 작성과 관리를 간단하고 강력하며 즐겁게 만들어주는 현대적인 문서화 프레임워크 **BaiDocs**에 오신 것을 환영합니다.

## 배울 내용

이 가이드는 다음을 도와드립니다:

- ✅ BaiDocs가 무엇이며 왜 사용해야 하는지 이해하기
- ✅ 첫 문서 프로젝트 설정하기
- ✅ MDX를 사용하여 아름다운 콘텐츠 작성하기
- ✅ 테마와 외관 커스터마이징하기
- ✅ 문서에서 PDF 생성하기
- ✅ 문서 사이트 배포하기

## 빠른 시작

빨리 시작하고 싶다면 가장 빠른 방법은 다음과 같습니다:

```bash
# 저장소 클론
git clone https://github.com/lablup/baidocs.git

# 의존성 설치
pnpm install

# 개발 서버 시작
pnpm dev
```

`http://localhost:3000`을 방문하면 실행 중인 문서를 볼 수 있습니다!

:::tip
이 전체 가이드는 BaiDocs 자체로 만들어졌으며, 자신의 문서에서 사용할 수 있는 모든 기능을 보여줍니다.
:::

## BaiDocs는 누구를 위한 것인가요?

BaiDocs는 다음과 같은 분들에게 완벽합니다:

- **소프트웨어 팀**: 포괄적인 API 문서와 사용자 가이드 작성
- **오픈소스 프로젝트**: 프로젝트를 위한 아름다운 문서 사이트 구축
- **기술 작가**: 기술 콘텐츠를 위한 현대적이고 유연한 플랫폼 사용
- **교육자**: 강좌 자료와 학습 리소스 제작

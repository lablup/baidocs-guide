# BaiDocs 가이드

BaiDocs는 문서의 생성, 관리, 게시를 가능한 한 원활하게 만들기 위해 설계된 현대적이고 강력한 문서화 프레임워크입니다. 현대 개발팀의 요구사항을 염두에 두고 구축된 BaiDocs는 마크다운의 간단함과 다국어 지원, 버전 제어, 자동화된 배포와 같은 고급 기능을 결합합니다.

## BaiDocs 소개

BaiDocs는 두 가지 주요 구성 요소로 이루어진 포괄적인 문서화 솔루션입니다:

- **에디터**: 문서 작성 및 관리를 위한 웹 기반 인터페이스
- **뷰어**: 사용자를 위한 빠르고 반응형인 문서 사이트

## 주요 기능

### 🎯 **간편한 콘텐츠 작성**
- 확장 문법을 사용한 마크다운/MDX 작성
- 라이브 미리보기가 포함된 비주얼 에디터
- 어드모니션, 코드 블록 등을 위한 내장 지원

### 🌍 **다국어 지원**
- 여러 언어에 대한 네이티브 지원
- 간편한 콘텐츠 번역 워크플로우
- 언어별 내비게이션

### 📚 **버전 제어 통합**
- 브랜치 지원이 포함된 Git 기반 워크플로우
- Git 저장소로부터 자동 배포
- 익숙한 Git 워크플로우를 통한 협업

### 🚀 **현대적 아키텍처**
- Next.js로 구축된 빠르고 반응형인 뷰어
- 자동 모노레포 감지
- 유연한 배포 옵션 (로컬, Vercel 등)

### 📄 **가져오기 및 변환**
- Sphinx, MkDocs, GitBook에서 가져오기
- BaiDocs 형식으로 자동 변환
- 가져오기 후 깔끔한 저장소 구조

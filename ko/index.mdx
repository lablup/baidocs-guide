# BaiDocs 가이드

BaiDocs에 오신 것을 환영합니다! 현대적이고 강력한 문서화 플랫폼으로 아름답고 기능이 풍부한 문서를 쉽게 만들 수 있습니다.

## BaiDocs란 무엇인가요?

BaiDocs는 개발자와 기술 작성자를 위해 설계된 차세대 문서화 플랫폼입니다. MDX (Markdown + JSX)를 기본으로 하여 간단한 텍스트 작성과 강력한 상호작용 구성요소를 결합합니다.

## 빠른 시작

### 1. 첫 번째 책 만들기

BaiDocs를 시작하는 가장 빠른 방법은 새로운 책을 만드는 것입니다:

:::tip 새 사용자이신가요?
[시작하기 개요](getting-started/overview.mdx)에서 BaiDocs 아키텍처와 핵심 개념을 먼저 이해해보세요.
:::

```bash
# BaiDocs 편집기 실행
npm run dev:editor

# 브라우저에서 http://localhost:3001 방문
# "새 책 만들기" 클릭하고 안내를 따르세요
```

## MDX의 강력함

일반적인 Markdown을 작성할 수 있습니다:

- **굵은 텍스트**와 *기울임 텍스트*
- [링크](https://baidocs.com)와 `인라인 코드`
- 목록과 표

그리고 React 컴포넌트도 사용할 수 있습니다:

## 주요 기능

### 📝 강력한 편집 경험

- **실시간 미리보기**: 변경사항을 즉시 확인
- **파일 탐색기**: 직관적인 콘텐츠 관리
- **Git 통합**: 버전 관리와 협업
- **자동 저장**: 작업 내용 자동 보존

### 🎨 아름다운 문서

- **반응형 디자인**: 모든 기기에서 완벽한 표시
- **다크 모드**: 사용자 선호도에 따른 자동 전환
- **구문 강조**: 코드 블록의 아름다운 표시
- **수학 지원**: LaTeX 수식 렌더링

### 🔍 지능형 검색

- **전체 텍스트 검색**: 모든 콘텐츠에서 즉시 검색
- **다국어 검색**: 언어별 검색 최적화
- **결과 하이라이트**: 검색어 강조 표시
- **키보드 내비게이션**: 접근성을 위한 완전한 키보드 지원

### 🌍 글로벌 도달

- **다국어 콘텐츠**: 무제한 언어 지원
- **CDN 배포**: 전 세계 빠른 로딩
- **SEO 최적화**: 검색 엔진 친화적
- **접근성**: WCAG 2.1 AA 준수

## 사용 사례

BaiDocs는 다양한 문서화 요구사항에 완벽합니다:

### 📖 제품 문서
- 사용자 가이드와 튜토리얼
- API 참조 문서
- 변경 사항 로그와 릴리스 노트

### 🏗️ 기술 문서
- 소프트웨어 아키텍처 문서
- 개발자 가이드와 모범 사례

### 🎓 교육 콘텐츠
- 강의 자료와 교과서
- 상호작용 튜토리얼
- 학습 경로와 진행 추적

## 무엇이 다른가요?

BaiDocs가 다른 문서화 도구들과 차별화되는 점:

### vs. 전통적인 정적 사이트 생성기

- **더 쉬운 설정**: 복잡한 구성 없이 즉시 시작
- **실시간 편집**: 즉각적인 피드백과 미리보기
- **고급 기능**: PDF, 검색, 다국어가 기본 제공

### vs. 호스팅된 문서 플랫폼

- **데이터 소유권**: Git으로 콘텐츠 완전 제어
- **사용자 정의**: 무제한 테마와 기능 확장
- **비용 효율성**: 오픈소스로 저렴한 운영

### vs. 위키 시스템

- **현대적 UX**: 아름답고 반응형 인터페이스
- **개발자 친화적**: 마크다운과 Git 워크플로
- **성능**: 빠른 로딩과 검색

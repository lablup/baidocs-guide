# 주요 기능

BaiDocs는 현대적인 문서화 워크플로 및 엔터프라이즈 요구사항을 위해 설계된 포괄적인 기능 세트를 제공합니다.

## 콘텐츠 관리

### 고급 마크다운 처리
- **MDX 통합**: 마크다운 콘텐츠 내에서 React 컴포넌트 임베딩
- **문법 확장**: 테이블, 경고문 및 사용자 정의 요소 지원
- **코드 강조**: 100개 이상의 프로그래밍 언어에 대한 구문 강조
- **수학 렌더링**: KaTeX 통합을 통한 LaTeX 수학 표현식

### 다국어 지원
- **동기화된 네비게이션**: 모든 언어 버전에서 일관된 구조
- **콘텐츠 조정**: 누락된 번역의 자동 감지
- **언어 전환**: 지속적인 네비게이션 상태를 유지하는 원활한 사용자 경험
- **RTL 언어 지원**: 우에서 좌로 읽는 텍스트 렌더링 기능

### 파일 관리
- **디렉토리 구조**: 가상 경로 매핑을 통한 계층적 구성
- **자산 관리**: 이미지, 비디오 및 문서의 중앙 집중식 처리
- **메타 폴더**: 공유 리소스를 위한 특수 디렉토리
- **자동 최적화**: 이미지 압축 및 형식 변환

## 버전 관리 통합

### Git 워크플로
- **브랜치 관리**: 기능 브랜치 및 릴리스 워크플로 지원
- **충돌 해결**: 내장된 병합 충돌 감지 및 해결 도구
- **커밋 조정**: 설명적인 메시지를 포함한 자동화된 커밋 생성
- **원격 동기화**: 인증 처리를 포함한 push 및 pull 작업

### 모노레포 지원
- **가상 루트 매핑**: 모노레포 구조에 대한 투명한 경로 변환
- **스파스 체크아웃**: 문서 하위 디렉토리의 선택적 복제
- **경로 격리**: 공유 저장소 내의 안전한 작업 경계
- **브랜치 필터링**: 문서 콘텐츠와 관련된 브랜치만 표시

## 사용자 인터페이스

### 편집기 경험
- **라이브 미리보기**: 마크다운 콘텐츠의 실시간 렌더링
- **분할 보기**: 나란히 편집 및 미리보기 모드
- **네비게이션 관리**: 콘텐츠 구조를 위한 시각적 트리 편집기
- **파일 작업**: 포괄적인 파일 및 디렉토리 관리 도구

### 뷰어 경험
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 디바이스에서의 최적 표시
- **검색 기능**: 고급 필터링 옵션을 포함한 전문 텍스트 검색
- **테마 사용자 정의**: 다양한 색상 구성 및 레이아웃 옵션
- **인쇄 최적화**: 페이지 나누기가 최적화된 CSS 인쇄

## 고급 기능

### PDF 생성
- **챕터 기반 내보내기**: 자동 문서 분할
- **사용자 정의 표지**: 구성 가능한 제목 페이지 및 브랜딩
- **목차**: 자동 생성된 네비게이션
- **이미지 최적화**: 인쇄를 위한 해상도 및 품질 최적화

### 검색 및 발견
- **전문 텍스트 인덱싱**: 포괄적인 콘텐츠 검색 기능
- **패싯 검색**: 콘텐츠 유형, 언어 및 메타데이터별 필터링
- **검색 강조**: 일치하는 용어의 시각적 강조
- **검색 분석**: 사용량 추적 및 최적화 인사이트

### 성능 최적화
- **정적 사이트 생성**: 최적 로딩 속도를 위한 사전 렌더링된 페이지
- **증분 빌드**: 수정된 콘텐츠의 선택적 재빌드
- **자산 최적화**: 자동 압축 및 캐싱 전략
- **CDN 통합**: 글로벌 콘텐츠 전송 네트워크 지원

## 엔터프라이즈 기능

### 보안 및 권한
- **액세스 제어**: 콘텐츠 관리를 위한 역할 기반 권한
- **인증 통합**: SSO 및 OAuth 공급자 지원
- **감사 로깅**: 포괄적인 활동 추적 및 규정 준수
- **콘텐츠 검증**: 자동화된 품질 확인 및 스타일 강제 적용

### 배포 및 호스팅
- **다양한 호스팅 옵션**: 정적 사이트, 서버리스 및 컨테이너 배포
- **CI/CD 통합**: 자동화된 빌드 및 배포 파이프라인
- **환경 관리**: 개발, 스테이징 및 프로덕션 워크플로
- **분석 통합**: 트래픽 모니터링 및 사용자 행동 추적

### 사용자 정의 및 확장성
- **플러그인 아키텍처**: 사용자 정의 컴포넌트 및 기능 확장
- **테마 개발**: 완전한 시각적 사용자 정의 기능
- **API 통합**: 외부 데이터 소스 및 동적 콘텐츠
- **웹훅 지원**: 외부 시스템과의 이벤트 기반 통합

이러한 기능들은 전체적으로 엔터프라이즈급 문서 관리 및 협업 워크플로를 위한 견고한 기반을 제공합니다.

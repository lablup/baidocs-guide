## Quick Start

If you're eager to get started, here's the fastest way:

```bash
# Clone the repository
git clone https://github.com/lablup/baidocs.git

# Install dependencies
pnpm install

# Start the development server
pnpm dev
```

Visit `http://localhost:3000` to see your documentation in action!

:::tip
This entire guide is built with BaiDocs itself, showcasing all the features you can use in your own documentation.
:::

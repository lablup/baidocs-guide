# MDX Features

MDX (Markdown + JSX) is the heart of BaiDocs content creation. It combines the simplicity of Markdown with the power of React components, allowing you to create rich, interactive documentation.

## What is MDX?

MDX allows you to write JSX directly in your Markdown files. This means you can:

- Use React components in your documentation
- Create custom layouts and styling
- Embed interactive elements
- Build reusable content blocks

```mdx
# Regular Markdown Header

This is regular markdown content with **bold** text.

<div style={{backgroundColor: '#f0f8ff', padding: '1rem', borderRadius: '8px'}}>
  This is a JSX component with custom styling!
</div>

import CustomComponent from '../components/CustomComponent'

<CustomComponent data="example" />
```

## JSX in MDX

### Inline JSX Elements

You can use any HTML/JSX element directly in your content:

```mdx
<div className="highlight-box">
  This content is in a custom container.
</div>

<span style={{color: 'red', fontWeight: 'bold'}}>
  Important red text
</span>

<details>
  <summary>Click to expand</summary>
  <p>Hidden content that can be toggled.</p>
</details>
```

**Result:**

<div style={{backgroundColor: '#f0f8ff', padding: '1rem', borderRadius: '8px'}}>
  This content is in a custom container.
</div>

<span style={{color: 'red', fontWeight: 'bold'}}>
  Important red text
</span>

### JavaScript Expressions

Use JavaScript expressions within your content:

```mdx
Current date: {new Date().toLocaleDateString()}

Math calculation: {2 + 3 = {2 + 3}}

Conditional content: {true ? 'This shows' : 'This does not show'}

{/* This is a JSX comment */}
```

**Result:**
Current date: {new Date().toLocaleDateString()}

Math calculation: 2 + 3 = {2 + 3}

```mdx
export const Tabs = ({ children, defaultTab = 0 }) => {
  const [activeTab, setActiveTab] = useState(defaultTab);
  const tabs = React.Children.toArray(children);

  return (
    <div style={{border: '1px solid #ddd', borderRadius: '8px'}}>
      <div style={{display: 'flex', borderBottom: '1px solid #ddd'}}>

          <button
            key={index}
            onClick={() => setActiveTab(index)}

              backgroundColor: activeTab === index ? '#f0f0f0' : 'transparent',
              cursor: 'pointer'

          >
            {tab.props.title}
          </button>
        ))}
      </div>
      <div style={{padding: '1rem'}}>
        {tabs[activeTab]}
      </div>
    </div>
  );
};

## Code Enhancement

### Code with Copy Button

Create code blocks with copy functionality:

```mdx
export const CodeBlock = ({ children, language, copyable = true }) => {
  const [copied, setCopied] = useState(false);

  const copyCode = () => {
    navigator.clipboard.writeText(children);
    setCopied(true);
    setTimeout(() => setCopied(false), 2000);
  };

  return (
    <div style={{position: 'relative'}}>
      {copyable && (
        <button
          onClick={copyCode}
          style={{
            position: 'absolute',
            top: '0.5rem',
            right: '0.5rem',
            padding: '0.25rem 0.5rem',
            fontSize: '0.8rem',
            backgroundColor: copied ? '#4caf50' : '#2196f3',
            color: 'white',
            border: 'none',
            borderRadius: '4px',
            cursor: 'pointer'
          }}
        >
          {copied ? 'Copied!' : 'Copy'}
        </button>
      )}
      <pre style={{
        backgroundColor: '#f5f5f5',
        padding: '1rem',
        borderRadius: '8px',
        overflow: 'auto'
      }}>
        <code>{children}</code>
      </pre>
    </div>
  );
};

### Highlighted Code Lines

Emphasize specific lines in code blocks:

```mdx
export const HighlightedCode = ({ children, highlightLines = [] }) => {
  const lines = children.split('\n');

  return (
    <pre style={{backgroundColor: '#f5f5f5', padding: '1rem', borderRadius: '8px'}}>
      {lines.map((line, index) => (
        <div
          key={index}
          style={{
            backgroundColor: highlightLines.includes(index + 1) ? '#fff3cd' : 'transparent',
            padding: '0 0.5rem',
            margin: '0 -0.5rem'
          }}
        >
          <code>{line}</code>
        </div>
      ))}
    </pre>
  );
};

## Best Practices for MDX

### Component Organization

Structure your components logically:

```mdx
// At the top of your file
import { Button, Alert, CodeBlock } from '../components'

// Define simple components inline
export const Highlight = ({ children, color = 'yellow' }) => (
  <mark style={{backgroundColor: color, padding: '0.1rem 0.2rem'}}>
    {children}
  </mark>
)

// Use components throughout your content
<Alert type="warning">
  Make sure to <Highlight color="orange">backup your data</Highlight> before proceeding.
</Alert>
```

### Performance Considerations

**Lazy Load Heavy Components**:
```mdx
import { lazy, Suspense } from 'react'

const HeavyChart = lazy(() => import('../components/HeavyChart'))

<Suspense fallback={<div>Loading chart...</div>}>
  <HeavyChart data={chartData} />
</Suspense>
```

**Optimize Inline Styles**:
```mdx
// Good: Define styles once
export const cardStyle = {
  border: '1px solid #ddd',
  borderRadius: '8px',
  padding: '1rem',
  marginBottom: '1rem'
}

<div style={cardStyle}>Content</div>
<div style={cardStyle}>More content</div>

// Avoid: Inline styles everywhere
<div style={{border: '1px solid #ddd', borderRadius: '8px'}}>Content</div>
```

### Accessibility

Make your components accessible:

```mdx
export const AccessibleButton = ({ children, onClick, ...props }) => (
  <button
    onClick={onClick}
    style={{
      padding: '0.75rem 1.5rem',
      fontSize: '1rem',
      border: '2px solid #2196f3',
      backgroundColor: '#2196f3',
      color: 'white',
      borderRadius: '8px',
      cursor: 'pointer'
    }}
    // Accessibility attributes
    role="button"
    aria-label={props.ariaLabel}
    tabIndex={0}
    {...props}
  >
    {children}
  </button>
)

<AccessibleButton
  ariaLabel="Download the installation guide"
  onClick={() => downloadGuide()}
>
  Download Guide
</AccessibleButton>
```

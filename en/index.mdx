# Welcome to BaiDocs

BaiDocs is a modern, powerful documentation framework designed to make creating, managing, and publishing documentation as seamless as possible. Built with the needs of modern development teams in mind, BaiDocs combines the simplicity of Markdown with advanced features like multi-language support, version control, and automated deployments.

## What is BaiDocs?

BaiDocs is a comprehensive documentation solution that consists of two main components:

- **Editor**: A web-based interface for creating and managing documentation
- **Viewer**: A fast, responsive documentation site for your users
